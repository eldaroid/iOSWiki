[–°—Ç–∞–Ω–¥–∞—Ä—Ç](https://refspecs.linuxfoundation.org/fhs.shtml) –æ–ø–∏—Å—ã–≤–∞–µ—Ç —Å–ª–µ–¥—É—é—â–∏–µ –¥–∏—Ä–µ–∫—Ç–æ—Ä–∏–∏ –≤ –∫–æ—Ä–Ω–µ–≤–æ–º –∫–∞—Ç–∞–ª–æ–≥–µ (/). `tree -d -L 1`:

* **/bin** - –û—Å–Ω–æ–≤–Ω—ã–µ [—Å–∏—Å—Ç–µ–º–Ω—ã–µ —É—Ç–∏–ª–∏—Ç—ã —Ç–µ—Ä–º–∏–Ω–∞–ª–∞ (CLI)](/1%20Common/1.3%20Terminal/1.3.3%20CLI/), –Ω–µ–æ–±—Ö–æ–¥–∏–º—ã–µ –¥–ª—è —Ä–∞–±–æ—Ç—ã –û–°;

* **/boot** - –§–∞–π–ª—ã, –Ω–µ–æ–±—Ö–æ–¥–∏–º—ã–µ –¥–ª—è –∑–∞–≥—Ä—É–∑–∫–∏ —Å–∏—Å—Ç–µ–º—ã, –≤–∫–ª—é—á–∞—è –∑–∞–≥—Ä—É–∑—á–∏–∫ –∏ —è–¥—Ä–æ.
* **/dev** - –§–∞–π–ª—ã —É—Å—Ç—Ä–æ–π—Å—Ç–≤, –æ–±–µ—Å–ø–µ—á–∏–≤–∞—é—â–∏–µ –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å –¥–ª—è –≤–∑–∞–∏–º–æ–¥–µ–π—Å—Ç–≤–∏—è —Å –æ–±–æ—Ä—É–¥–æ–≤–∞–Ω–∏–µ–º –∫–æ–º–ø—å—é—Ç–µ—Ä–∞.
* **/etc** - –ö–æ–Ω—Ñ–∏–≥—É—Ä–∞—Ü–∏–æ–Ω–Ω—ã–µ —Ñ–∞–π–ª—ã —Å–∏—Å—Ç–µ–º—ã (–Ω–∞—Å—Ç—Ä–æ–π–∫–∏).


* **/home** - –î–æ–º–∞—à–Ω–∏–µ –¥–∏—Ä–µ–∫—Ç–æ—Ä–∏–∏ –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª–µ–π, –∫—Ä–æ–º–µ `root` (–¥–ª—è –Ω–µ–≥–æ –æ—Ç–¥–µ–ª—å–Ω–∞—è –ø–∞–ø–∫–∞).

* **/media** - –¢–æ—á–∫–∏ –º–æ–Ω—Ç–∏—Ä–æ–≤–∞–Ω–∏—è –¥–ª—è —Å—ä–µ–º–Ω—ã—Ö –Ω–æ—Å–∏—Ç–µ–ª–µ–π (–Ω–∞–ø—Ä–∏–º–µ—Ä, CD-ROM, USB).
* **/opt** - –î–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω–æ–µ –ø—Ä–æ–≥—Ä–∞–º–º–Ω–æ–µ –æ–±–µ—Å–ø–µ—á–µ–Ω–∏–µ, –∫–æ—Ç–æ—Ä–æ–µ —É—Å—Ç–∞–Ω–∞–≤–ª–∏–≤–∞–µ—Ç—Å—è –≤ –≤–∏–¥–µ –ø–∞–∫–µ—Ç–æ–≤.


* **/mnt** - –¢–æ—á–∫–∏ –≤—Ä–µ–º–µ–Ω–Ω–æ–≥–æ –º–æ–Ω—Ç–∏—Ä–æ–≤–∞–Ω–∏—è —Ñ–∞–π–ª–æ–≤—ã—Ö —Å–∏—Å—Ç–µ–º.

* **/tmp** - –í—Ä–µ–º–µ–Ω–Ω—ã–µ —Ñ–∞–π–ª—ã, –∫–æ—Ç–æ—Ä—ã–µ –º–æ–≥—É—Ç –±—ã—Ç—å —É–¥–∞–ª–µ–Ω—ã –ø—Ä–∏ –ø–µ—Ä–µ–∑–∞–≥—Ä—É–∑–∫–µ.
* **/run** - –í—Ä–µ–º–µ–Ω–Ω—ã–µ —Ñ–∞–π–ª—ã, –∫–æ—Ç–æ—Ä—ã–µ —Å–æ–∑–¥–∞—é—Ç—Å—è –∏ –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è –≤–æ –≤—Ä–µ–º—è —Ä–∞–±–æ—Ç—ã —Å–∏—Å—Ç–µ–º—ã.

* **/usr** - –í—Ç–æ—Ä–∏—á–Ω–∞—è –∏–µ—Ä–∞—Ä—Ö–∏—è –¥–ª—è –¥–∞–Ω–Ω—ã—Ö –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—è; –≤–∫–ª—é—á–∞–µ—Ç –±–æ–ª—å—à–∏–Ω—Å—Ç–≤–æ –º–Ω–æ–≥–æ–ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏—Ö –∫–æ–º–∞–Ω–¥ –∏ –¥–∞–Ω–Ω—ã—Ö. –£—Å—Ç–∞–Ω–∞–≤–ª–∏–≤–∞–µ–º–æ–µ –ü–û –±—É–¥–µ—Ç –ø–∞–¥–∞—Ç—å –∏–º–µ–Ω–Ω–æ —Å—é–¥–∞.
* **/usr/bin or /usr/sbin** - –ê–Ω–∞–ª–æ–≥–∏—è —Å /bin –∏ /sbin, –Ω–æ –Ω–∞–º /usr/bin –Ω–µ –∫—Ä–∏—Ç–∏—á–Ω—ã –¥–ª—è –∑–∞–ø—É—Å–∫–∞ –û–°, –ø—Ä–µ–¥–Ω–∞–∑–Ω–∞—á–µ–Ω—ã —Å–∫–æ—Ä–µ–µ –¥–ª—è –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª–µ–π (–Ω–µ –û–°);
* **c** - –≤—ã –∫–æ–º–ø–∏–ª–∏—Ä—É–µ—Ç–µ –ø—Ä–æ–≥—Ä–∞–º–º—É —Å–∞–º–æ—Å—Ç–æ—è—Ç–µ–ª—å–Ω–æ/—Å–∫–∞—á–∏–≤–∞–µ—Ç–µ —Å—Ç–æ—Ä–æ–Ω–Ω—é—é —É—Ç–∏–ª–∏—Ç—É. –ù/—Ä: wget, python3, docker –∏ git;
* **/var** - –ü–µ—Ä–µ–º–µ–Ω–Ω—ã–µ —Ñ–∞–π–ª—ã, —Ç–∞–∫–∏–µ –∫–∞–∫ –∂—É—Ä–Ω–∞–ª—ã, –≤—Ä–µ–º–µ–Ω–Ω—ã–µ —Ñ–∞–π–ª—ã –∏ –ø–æ—á—Ç–∞ (–∏–∑–º–µ–Ω—è–µ–º—ã–π —Ñ–∞–π–ª—ã –ø—Ä–æ–≥—Ä–∞–º–º).
