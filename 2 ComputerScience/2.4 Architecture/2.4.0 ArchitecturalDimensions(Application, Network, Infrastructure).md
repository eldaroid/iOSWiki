# Architectural Dimensions: Application, Network, Infrastructure

1-tier, 2-tier, 3-tier, N-tier?

## Application Architecture

* Monolithic — всё в одном

* Microservices — множество мелких сервисов

* Сервис-ориентированная (Service-Oriented, SOA) — крупные сервисы с шиной

## Network interaction architecture

* Client-Server (Клиент-Сервер)

* Peer-to-Peer (P2P, Одноранговая)

* Event-Driven (Событийно-ориентированная)

## Infrastructure Architecture

* Web Application (Веб-приложение)

* Web Server (Веб-сервер)

* Application Server (Сервер приложений)
