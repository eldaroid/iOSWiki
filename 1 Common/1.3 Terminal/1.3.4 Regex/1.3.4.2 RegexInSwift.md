## Regex In Swift

### Пример

Все отрицательные числа

```swift
let input = "0 12 -5 123 -18 -7"
// regex pattern
let regex = /-\d+/
let res = input.matches(of: regex)
for i in res.indices {
    print(res[i].output)
}
```
